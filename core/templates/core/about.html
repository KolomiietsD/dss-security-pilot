Про систему

Information Security DSS — навчальний проєкт для підтримки рішень у сфері кібербезпеки.

Що це за проєкт?

Ця система розробляється як Decision Support System (DSS) для фахівців з інформаційної безпеки. Вона допомагає аналізувати події безпеки, ризики та стан інфраструктури, комбінуючи дані з різних джерел.

Основна ідея — надати зручний інтерфейс, де можна побачити: агреговану інформацію по хостах, детекти з CrowdStrike, а в майбутньому — результати роботи моделей машинного навчання.

Модуль
CrowdStrike

Отримання інформації про хости, детекти та події безпеки з платформи CrowdStrike. Використовується як основне джерело технічних даних про інциденти.

Модуль
Wazuh (SIEM / XDR)

Інтеграція з Wazuh для збору логів, подій безпеки та стану агентів. Доповнює дані CrowdStrike контекстом із хостів і лог-платформи, дозволяючи краще бачити картину інцидентів.

Модуль
Персептрон

Експериментальний модуль з машинного навчання, який у майбутньому обчислюватиме коефіцієнт ризику для хостів та користувачів на основі історичних даних.

Фокус
Підтримка рішень

Система не замінює аналітика, а допомагає йому: швидше побачити критичні вузли, оцінити ризик та прийняти більш виважене рішення щодо реагування.

Мета розробки

Проєкт створюється як частина навчальної / дослідницької роботи у сфері інформаційної безпеки та data science. Він демонструє, як поєднати дані з реальної системи захисту (CrowdStrike) з аналітичними модулями та моделями машинного навчання.

Подальші кроки включають розширення інтеграцій, покращення UX, а також впровадження повноцінної моделі оцінки ризику.
